Welcome to Rapid7 Cookbooks.

Cookbooks

  • DRb - A cookbook which will, by default, configure a DRb server to be vulnerable to Metasploit's drb_remote_codeexec module.
  • DVWA - A cookbook to install Damn Vulnerable Web App in either standalone form or over XAMPP.
  • Mirror - Configures a node as a git/hg repository mirror
  • Unsafe Umask - Configures Ubuntu to have an unsafe umask
  • WAVSEP - Recipes to install the Web Application Vulnerability Scanner Evaluation Project and run a web server using Tomcat
  • Windows Security - Recipes to configure ASLR, DEP, and other Windows configuration options
  • XAMPP - Recipes to install the XAMPP web server on a node
